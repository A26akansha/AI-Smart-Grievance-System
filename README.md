# AI Smart Grievance System

## Overview
AI Smart Grievance System is a web-based complaint management application developed using Flask and SQLite. The system allows users to register, log in, submit complaints, and track complaint status.

## Features
- User Registration and Login
- Complaint Submission
- AI-based Complaint Category Detection
- AI-based Priority Detection
- Admin Panel
- Complaint Status Management
- Dashboard Analytics
- SQLite Database Integration

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
